## Goal

Create a working TypeScript project skeleton that matches the directory layout in `project-structure.md`, passes `tsc --noEmit` and `biome check`, and can run a minimal "hello" CLI entry point.

## Scope

### In scope
- `package.json` with all declared dependencies from ADR-0001.
- `pnpm-lock.yaml` generated by `pnpm install`.
- `tsconfig.json` with all strict settings from `coding-standards.md ┬ž1`.
- `tsup.config.ts` for build output.
- `vitest.config.ts` for test runner.
- `biome.json` with lint + format config.
- Directory stubs: `src/cli/`, `src/core/`, `src/adapters/`, `src/util/`, `tests/`, `bin/`.
- Minimal `src/cli/index.ts` entry point that registers a `--version` flag and prints the version.
- `bin/skillrelay` shim.
- `.gitignore` updated to include `dist/`, `node_modules/`.

### Out of scope
- Any domain logic (schema, registry, adapters).
- CI workflows (added later).
- `.npmignore` (added before publish).
